🧠 Lesson 3: Think Like a Game Designer!

Goal: Break down the Space Invader game into logical steps and understand what an algorithm is.

🧩 Can You Break It Down?

What are the main pieces of our game?

  • 🚀 The spaceship (player)
  • 🔫 The bullets (fired with spacebar)
  • 👾 The enemies (that move and drop down)
  • 🎯 The score (that increases when you hit an enemy)

📋 What Are the Steps?

Can you imagine what instructions the game needs to follow to work?

This is called an algorithm — a step-by-step list of what to do!
  1. Start the game and show the screen
  2. Wait for the player to press keys
  3. If the player presses ➡️ or ⬅️, move the spaceship
  4. If the player presses spacebar, fire a bullet
  5. Move the enemies across the screen
  6. Check if any bullet hits an enemy
  7. If yes, remove both and increase score
  8. If all enemies are gone, start again!

✍️ Try It Yourself!

Can you write your own algorithm for this game in your notebook?

Start with “When the game starts…” and think about what happens next.
