🧱 Lesson 4: Let’s Start Coding!

Now that you've explored the game and thought about the steps, it's time to actually start building your own version of Space Invader!

👩‍💻 Let’s Set Up Our Game Window

We’ll use Python and Pygame to create a screen where the game will run.

import pygame

# Initialize the game engine
pygame.init()

# Create the game window
screen = pygame.display.set_mode((800, 600))
pygame.display.set_caption("Space Invader")
🧠 Key Terms:
pygame.init() – starts the Pygame engine
set_mode() – creates the game screen
set_caption() – sets the title of the window

🚀 Add the Player (Spaceship)

We’ll load the image and place it at the bottom of the screen.

# Load the player image
player_img = pygame.image.load("player.png")
player_x = 368
player_y = 500

# Draw the player
def draw_player(x, y):
    screen.blit(player_img, (x, y))
screen.blit() – draws the image on the screen

🎮 Add Movement

We want the spaceship to move left and right when you press arrow keys.

player_speed = 0

# Inside your game loop:
for event in pygame.event.get():
    if event.type == pygame.KEYDOWN:
        if event.key == pygame.K_LEFT:
            player_speed = -0.5
        if event.key == pygame.K_RIGHT:
            player_speed = 0.5
    if event.type == pygame.KEYUP:
        if event.key in [pygame.K_LEFT, pygame.K_RIGHT]:
            player_speed = 0

# Update player position
player_x += player_speed

🔫 Fire a Bullet!

We’ll shoot a bullet when the player presses the spacebar.

# Load the bullet image
bullet_img = pygame.image.load("bullet.png")
bullet_x = 0
bullet_y = player_y
bullet_speed = 1.5
bullet_state = "ready"

def fire_bullet(x, y):
    global bullet_state
    bullet_state = "fire"
    screen.blit(bullet_img, (x + 28, y - 20))

# Inside your event loop:
if event.type == pygame.KEYDOWN:
    if event.key == pygame.K_SPACE:
        if bullet_state == "ready":
            bullet_x = player_x
            fire_bullet(bullet_x, bullet_y)
