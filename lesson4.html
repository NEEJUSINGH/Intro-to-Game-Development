Step 1: Start Pygame and Create a Window

This code sets up your game screen and gives it a title:

import pygame

# Initialize the game engine
pygame.init()

# Create the game window
screen = pygame.display.set_mode((800, 600))
pygame.display.set_caption("Space Invader")
Quick Tips:
pygame.init() ‚Äì starts the game engine
set_mode() ‚Äì sets the screen size
set_caption() ‚Äì names your game window
