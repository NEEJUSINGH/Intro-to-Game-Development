🛠️ Lesson 5: Power Up the Game!

In this final step: You’ll make enemies move, let bullets destroy them, and track your score!

1️⃣ Make Enemies Move and Bounce

We want the enemies to move side to side and bounce when they hit the wall.

let direction = 1;
onUpdate(() => {
 enemyGroup.forEach(enemy => {
  enemy.move(direction * ENEMY_SPEED, 0);
 });

 // If any enemy hits wall, bounce
 if (enemyGroup.some(e => e.pos.x <= 0 || e.pos.x + e.width >= width())) {
  direction *= -1;
  enemyGroup.forEach(e => e.pos.y += 20);
 }
});

2️⃣ Destroy Enemy on Bullet Hit

Let’s make bullets destroy enemies and add a kaboom effect.

onCollide("bullet", "enemy", (bullet, enemy) => {
 destroy(bullet);
 destroy(enemy);
 addKaboom(enemy.pos);
});

3️⃣ Add a Score Display

Show a score at the top and increase it when an enemy is hit.

let score = 0;
const scoreLabel = add([
 text("Score: 0"),
 pos(20, 20),
 layer("ui")
]);

onCollide("bullet", "enemy", (b, e) => {
 destroy(b);
 destroy(e);
 addKaboom(e.pos);
 score++;
 scoreLabel.text = "Score: " + score;
});

4️⃣ Win Condition

When all enemies are destroyed, show a message!

onUpdate(() => {
 if (enemyGroup.every(e => !e.exists())) {
  add([
   text("🎉 You Win!", { size: 40 }),
   pos(width() / 2 - 100, height() / 2),
  ]);
 }
});

Recap of Terms

  • onUpdate() – Runs continuously every frame
  • onCollide() – Runs when two things collide
  • destroy() – Removes something from the screen
  • add() – Adds an object to the screen
  • text() – Displays text

Final Tip

Your game doesn’t have to be perfect. What matters is that you understood how to build it step by step — like a game designer!
