Lesson 5: Move the Spaceship

Now let’s make the spaceship move when you press the arrow keys!

Step 1: Add Speed Control

player_speed = 0

Step 2: Handle Key Presses

Update your game loop with this logic:

# Inside your game loop:
for event in pygame.event.get():
    if event.type == pygame.KEYDOWN:
        if event.key == pygame.K_LEFT:
            player_speed = -0.5
        if event.key == pygame.K_RIGHT:
            player_speed = 0.5
    if event.type == pygame.KEYUP:
        if event.key in [pygame.K_LEFT, pygame.K_RIGHT]:
            player_speed = 0

Step 3: Move the Player

# Update player position
player_x += player_speed
player_x changes every frame, making it move left or right!

What You Learned

  • How to use pygame.KEYDOWN and KEYUP
  • How to make the player move using arrow keys
  • How to control speed with variables
